About

I enjoy working with computers and learning new things. I created this site as a summer project and to keep busy and strengthen my technology skills. Over the course of this project I've gained a familiarity with Amazon Web Services (EC2, Route 53, and S3). I've learned aspects of managing a Virtual Private Server remotely via SSH. I created my own domain name and configured DNS records; configured and installed Apache, MySQL, and PHP; and set up proper file permissions and restricting access to certain ports.

I've also learned a lot more about web development (HTML5,Javascript,JQuery). I love using Bootstrap -- it made styling this website so easy! It's also been nice using different tools such as Eclipse and Webstorm to develop and debug code efficiently. Since this website is somewhat simple I usually just worked in Notepad++ and emacs. Additionally learning about content management with Git was pretty awesome.

Anyway, that's enough about me. Thanks for stopping by!
