Resume Builder

WYSWG basic resume builder with drag and drop profile picture feature. Ready for printing on standard 8.5in x 11in paper. All editing can be done in the browser. Automatically saves every 3 seconds in case browser crashes or navigate away from the page. Data is stored locally in users browser and can be removed by clearing cache.
